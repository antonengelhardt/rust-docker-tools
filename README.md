# rust-docker-tools

This repository contains a set of tools to help with building and testing Rust projects in Docker containers.

The following tools are added to the base image [rust:1.80-alpine](https://hub.docker.com/_/rust)

* `curl`
* `rustfmt`
* `clippy`
* `musl-dev`
* `wasm32-wasi`-target
* `cargo-binutils`
* `perl`
* `make`
* `cargo-deny`
* `cargo-audit`
* `cargo-outdated`
